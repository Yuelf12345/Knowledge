# 常见题

## vue项目有那些常见的优化手段?
1. 代码层面
   - 使用v-show和v-if
   - keep-alive 缓存组件
   - 合理使用 key 属性
   - v-model.lazy
   - Object.freeze()
2. 资源优化
   - 图片压缩
   - 使用 import 路由懒加载
   - Suspence 组件 懒加载
3. 构建优化
   - Tree Shaking：移除未使用的代码
   - UglifyJS：压缩代码
   - SplitChunks：分割代码
4. 网络优化
   - 启用Gzip压缩，减少传输文件的大小
   - 使用 CDN 加速静态资源
   - 合理设置缓存策略
     - 强缓存：通过设置 Expires 和 Cache-Control 头部实现
     - 协商缓存：通过设置 Last-Modified 和 ETag 头部实现通过设置 Expires 和 Cache-Control 头部实现：

## 谈谈xss和CSFR
    XSS（Cross-Site Scripting，跨站脚本攻击）是一种利用网站漏洞，通过向用户显示恶意脚本来获取用户敏感信息的攻击方式。
    CSRF（Cross-Site Request Forgery，跨站请求伪造）是一种利用网站漏洞，通过向用户显示恶意链接或表单来获取用户敏感信息的攻击方式。
