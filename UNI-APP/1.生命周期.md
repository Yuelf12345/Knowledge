# UniApp的生命周期可以分为三个主要部分：应用生命周期、页面生命周期和组件生命周期。

## 应用生命周期
    应用生命周期函数定义在 App.vue 中，用于处理整个应用的启动、显示、隐藏等状态：
- onLaunch: 应用启动时触发，通常用于初始化数据、启动动画等。
- onShow: 应用显示时触发，通常用于恢复数据、恢复动画等。
- onHide: 应用隐藏时触发，通常用于保存数据、停止动画等。
### 以下事件 app-uvue 不支持
- onError: 应用发生错误时触发，通常用于处理错误信息。
- onUniNViewMessage: uni-app 运行在 H5 端时，uniNView 发送消息给 App 时触发。
- onUnhandledRejection: 应用发生未处理的 Promise 拒绝时触发，通常用于处理错误信息。
- onPageNotFound：应用打开页面不存在时触发，通常用于处理错误信息。
- onThemeChange：应用主题改变时触发，通常用于处理主题信息。
- onLastPageBackPress：应用返回上一页时触发，通常用于处理返回事件。 (app-uvue-android 3.9+)
- onExit：应用退出时触发，通常用于处理退出事件。(app-uvue-android 3.9+)

## 页面生命周期
    页面生命周期函数在每个页面中都可以使用，用于处理页面的加载、显示、隐藏等状态：
- onInit：页面初始化时触发，通常用于初始化页面数据。(3.1.0+)
- onLoad：页面加载时触发，参数为上一个页面传递的数据，通常用于初始化数据。
- onShow：页面显示时触发。
- onReady：页面初次渲染完成时触发，只触发一次，通常用于初始化数据。
- onHide：页面隐藏时触发。
- onUnload：页面卸载时触发，通常用于清理数据。
###  页面交互事件
- onResize：页面尺寸变化时触发，参数为变化后的尺寸。
- onPullDownRefresh：下拉刷新时触发，参数为当前页面的页面实例。
- onReachBottom：页面滚动到底部时触发，参数为当前页面的页面实例。
- onTabItemTap：点击 tab 栏按钮时触发，参数为点击的 tab 栏按钮的索引。
- onShareAppMessage：分享给好友时触发，参数为当前页面的页面实例。
- onPageScroll: 页面滚动时触发，参数为当前页面的页面实例。
- onNavigationBarButtonTap: 点击导航栏按钮时触发，参数为点击的导航栏按钮的索引。
- onBackPress: 返回键被点击时触发，参数为当前页面的页面实例。
- onNavigationBarSearchInputChanged: 搜索栏输入内容时触发，参数为输入的内容。
- onNavigationBarSearchInputConfirmed: 搜索栏输入内容并确认时触发，参数为输入的内容。
- onNavigationBarSearchInputClicked: 搜索栏被点击时触发，参数为点击的按钮。
- onShareTimeline: 分享到朋友圈时触发，返回一个对象，包含 title、desc、imageUrl。(2.8.1+)
- onAddToFavorites: 添加到收藏时触发，返回一个对象，包含 title、desc、imageUrl。 (2.8.1+)

## 组件生命周期
    组件生命周期与Vue的标准生命周期基本一致：
- beforeCreate: 组件实例被创建之前调用，此时 data 属性还没被定义。
- created: 组件实例被创建之后调用，此时 data 属性已被定义。
- beforeMount: 组件挂载之前调用，此时组件的 DOM 节点还没被渲染。
- mounted: 组件挂载之后调用，此时组件的 DOM 节点已被渲染。
- beforeUpdate: 组件更新之前调用，此时组件的 DOM 节点已被更新。
- updated: 组件更新之后调用，此时组件的 DOM 节点已被更新。
- beforeDestroy: 组件销毁之前调用，此时组件的 DOM 节点还没被销毁。
- destroyed: 组件销毁之后调用，此时组件的 DOM 节点已被销毁。

## 生命周期执行顺序
### 应用首次启动：
  1. App.onLaunch -> APP.show
  2. 页面.onLoad -> 页面.onShow
  3. Vue.beforeCreate -> Vue.created -> Vue.beforeMount -> Vue.mounted
  4. 页面.onReady
### 打开新页面:
  1. 旧页面.onHide
  2. 新页面.onLoad -> 新页面.onShow
  3. Vue.beforeCreate -> Vue.created -> Vue.beforeMount -> Vue.mounted
  4. 新页面.onReady
### 返回上一页
  1. 新页面.onUnload
  1. 旧页面.onShow
### 切到后台
  1. App.onHide
  2. PageA.onHide
### 切回前台
  1. 页面.onShow
  2. 页面.onShow
