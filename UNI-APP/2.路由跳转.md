# 页面跳转方式：通过 uni.navigateTo()、uni.redirectTo()、uni.reLaunch()、uni.switchTab() 方法跳转页面。

## uni.navigateTo()
- 作用：保留当前页面，跳转到应用内的某个页面
- 特点：可以返回原页面，会将跳转的页面加入页面栈

1. 调用api
```javascript
uni.navigateTo({
    url: '/pages/detail/detail?id=123', // 跳转的页面路径
    animationType: 'slide-in-right', // 跳转页面的动画效果
    animationDuration: 300,// 跳转页面的动画时长
    success: function(res) { }, // 跳转成功回调
    fail: function(res) { }, // 跳转失败回调
    complete: function(res) { } // 跳转完成回调
});
```
2. 使用组件
```vue
 <navigator open-type="navigate" url="/pages/detail?id=123">< />
```

## uni.redirectTo()
- 作用：关闭当前页面，跳转到应用内的某个页面
- 特点：不能返回原页面，将跳转的页面加入页面栈
1. 调用api
```javascript
uni.redirectTo({
    url: '/pages/detail/detail?id=123', // 跳转的页面路径
    success: function(res) { }, // 跳转成功回调
    fail: function(res) { }, // 跳转失败回调
    complete: function(res) { } // 跳转完成回调
});
```
2. 使用组件
```vue
 <navigator open-type="redirect" url="/pages/detail?id=123">< />
```
## uni.reLaunch()
- 作用：关闭所有页面，打开到应用内的某个页面
- 特点：重新加载整个应用的页面栈
1. 调用api
```javascript
uni.reLaunch({
    url: '/pages/detail/detail?id=123', // 跳转的页面路径
    success: function(res) { }, // 跳转成功回调
    fail: function(res) { }, // 跳转失败回调
    complete: function(res) { } // 跳转完成回调
});
```
2. 使用组件
```vue
 <navigator open-type="reLaunch" url="/pages/detail?id=123">< />
```
## uni.switchTab()
- 作用：跳转到 tabBar 页面，并关闭其他所有非 tabBar 页面
- 特点：只能跳转到配置在 tabBar 中的页面
1. 调用api
```javascript
uni.switchTab({
    url: '/pages/index/index', // 跳转的页面路径
    success: function(res) { }, // 跳转成功回调
    fail: function(res) { }, // 跳转失败回调
    complete: function(res) { } // 跳转完成回调
});
```
2. 使用组件
```vue
 <navigator open-type="switchTab" url="/pages/index/index">< />
```
## uni.navigateBack()
- 作用：关闭当前页面，返回上一页面
- 特点：返回上一页面，将跳转的页面加入页面栈
1. 调用api
```javascript
uni.navigateBack({
    delay: 0, // 返回的页面数，如果 delta 大于现有页面数，则返回到首页
    animationType: 'slide-in-right', // 跳转页面的动画效果
    animationDuration: 300,// 跳转页面的动画时长
    success: function(res) { }, // 跳转成功回调
    fail: function(res) { }, // 跳转失败回调
    complete: function(res) { } // 跳转完成回调
});
```
2. 使用组件
```vue
 <navigator open-type="navigateBack" url="/pages/index/index">< />
```
